# Data-Structures using C

This repository contains a comprehensive collection of C programs, systematically organized from basic concepts to advanced data structures and algorithms.

It is designed as a complete learning path for C programming, covering:

- Core programming fundamentals
- Functions and recursion
- Data structures (arrays, linked lists, stacks, queues)
- Advanced problem-solving and algorithms

## 🚀 How to Use

1. Clone the repository:

    ```bash
        git clone https://github.com/PrateekRaj8125/Data-Structures-using-C.git
        cd Data-Structures-using-C
    ```

2. Compile any program:

    ```bash
        gcc filename.c -o output
    ```

3. Run the program:

    ```bash
        ./output
    ```
