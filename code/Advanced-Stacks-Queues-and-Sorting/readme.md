# Advanced Stacks, Queues, and Sorting

This folder contains advanced C programs that extend stack and queue concepts and introduce basic sorting algorithms. It focuses on real-world applications of stacks and queues along with fundamental sorting techniques.

---

## 📚 Topics Covered

- Postfix expression evaluation using stack
- Duplicate parentheses detection
- Binary conversion using stack
- Deque (Double Ended Queue) using array and linked list
- Circular queue using linked list
- Sorting algorithms (Bubble Sort, Selection Sort)
