# Stacks and Expression Evaluation

This folder contains C programs based on the **stack data structure** and its applications. It includes implementations using both **arrays and linked lists**, along with real-world applications like **expression conversion and evaluation**.

---

## 📚 Topics Covered

- Stack implementation using arrays
- Stack implementation using linked lists
- Push and Pop operations
- Stack overflow and underflow
- Infix to postfix conversion
- Expression evaluation using stacks

## 🎯 Purpose

This collection helps in:

- Understanding stack operations and memory handling

- Building a foundation for compiler design and parsing
