# Searching and Advanced Algorithms

This folder contains C programs focused on **searching techniques, advanced sorting algorithms, and practical problem-solving using data structures**.

These programs help in building strong algorithmic foundations and improving efficiency in handling data.

---

## 📚 Topics Covered

- Linear Search (iterative and recursive)
- Binary Search
- Sorting algorithms (Insertion, Quick, Recursive Selection)
- String sorting (alphabetical order)
- Queue-based problem solving
- Linked list manipulation (duplicate removal)
