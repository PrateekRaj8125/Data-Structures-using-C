# Queues and Circular Queues

This folder contains C programs based on the **queue data structure**, including implementations using arrays, linked lists, and circular queues.

These programs demonstrate fundamental queue operations and variations, helping in understanding FIFO-based data handling.

---

## 📚 Topics Covered

- Queue implementation using arrays
- Queue implementation using linked lists
- Circular queue using linked list
- Insert (Enqueue) operation
- Delete (Dequeue) operation
- Display queue elements
- Queue overflow and underflow
- Checking if queue is empty
