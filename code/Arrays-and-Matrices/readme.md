# Arrays and Matrices

This folder contains C programs focused on **array and matrix operations**, including basic manipulations, searching, sparse matrix representation, and polynomial operations.

These programs help build a strong foundation in data structures using arrays and matrices.

---

## 📚 Topics Covered

- 1-D Array operations (insertion, deletion, searching, traversal)
- Sorted array insertion
- Matrix operations
- Sparse matrix representation (3-tuple format)
- Sparse matrix transpose
- Polynomial representation and addition
