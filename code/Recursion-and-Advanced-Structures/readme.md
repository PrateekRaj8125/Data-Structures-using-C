# Recursion and Advanced Structures

This folder contains C programs focused on **recursion and advanced structure concepts**. These programs demonstrate problem-solving using recursive techniques and advanced manipulation of structures such as passing structures by value, reference, and returning structures from functions.

---

## 📚 Topics Covered

- Recursion (array operations, GCD, Fibonacci, palindrome, reverse)
- Structure handling
- Passing structures by value and reference
- Returning structures from functions
- Nested structures
- Real-world data representation using structures
