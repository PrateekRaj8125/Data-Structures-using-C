# Functions-and-Structures

This folder contains C programs focused on functions and structures, covering important concepts such as call by value, call by address, recursion, and structured data handling using arrays of structures.

These programs are designed to strengthen problem-solving skills and understanding of modular programming in C.

---

## 📚 Topics Covered

- Function basics
- Call by value vs call by address
- Recursion
- Prime number checking
- Structures in C
- Arrays of structures
- Complex number operations
