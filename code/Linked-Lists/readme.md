# Linked Lists

This folder contains C programs based on **linked list data structures**, including both **single linked lists** and **doubly linked lists**. It demonstrates various operations such as insertion, traversal, searching, sorting, and reversing.

These programs help in understanding dynamic memory allocation and pointer-based data structures.

---

## 📚 Topics Covered

- Single Linked List (SLL)
- Doubly Linked List (DLL)
- Dynamic memory allocation
- Insertion operations
- Traversal
- Searching
- Sorting
- Reversing linked lists
