## 🛠 Development Setup

1. Fork the repository
2. Clone your fork:

    ```bash
        git clone https://github.com/PrateekRaj8125/Data-Structures-using-C.git
        cd C-Programming-Practice
    ```

3. Ensure you have a C compiler (GCC):

    ```bash
        gcc program.c -o program
        ./program
    ```

---

## ✅ Contribution Guidelines

- Follow clean coding practices:
  - Proper indentation
  - Meaningful variable names
  - Clear and concise comments
- Ensure:
  - Code compiles without errors
  - Output is correct
  - Logic is efficient (where applicable)

---

## 🐞 Reporting Issues

Use GitHub Issues and include:

- Clear description
- Steps to reproduce
- Expected vs actual behavior
- Screenshots/logs (if needed)
